# Superteam Malaysia Website

## Overview
Official community website for Superteam Malaysia, the local chapter of the global Superteam network for Solana builders in Malaysia.

## Status
Active (building for Superteam Earn bounty, 3,000 USDG)

## Tech Stack
- Next.js 16+ (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion + Lenis (smooth scroll)
- Supabase (PostgreSQL, Auth, Storage)
- Vercel (hosting)
- Luma API (event sync)

## Design Direction
"Tropical Futurism": Cinematic scroll-driven experience. Malaysian tropical nature meets Solana's tech-forward identity. AI-generated scenic imagery with botanical line-art overlays.

## Key URLs
- Bounty: superteam.fun/earn (Superteam Malaysia Website Design & Build Challenge)
- Contact: @SuperteamMY on X, t.me/SuperteamMY
- Ref sites: superteam.fun, uae.superteam.fun, de.superteam.fun

## Architecture
- Landing page: 10 sections (Hero, Mission, Stats, Events, Members, Partners, Wall of Love, FAQ, Join CTA, Footer)
- Members page: Searchable/filterable directory
- Admin dashboard: Supabase-powered CMS with CRUD for all content
- Luma integration: Cron-synced event data
- ISR with on-demand revalidation

## Fonts
- Satoshi (self-hosted, display/headings)
- Space Grotesk (next/font/google, body)
- Geist Mono (next/font/google, data/stats)
